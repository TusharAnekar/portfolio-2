# Portfolio

## Live Link - https://tushar-anekar.netlify.app/

A porfolio web application showing my projects and experience.

---

## INSTALLATION

```
1. git clone https://github.com/TusharAnekar/portfolio-2.git
2. cd portfolio-2
3. npm install
4. npm start
```

---

## FEATURES

1. Hero section
2. Skills section
3. Work Experience section
4. Blogs link
5. Contact section
6. Cart Management
7. Resume link
8. Dark/Light mode

---

## TECHNOLOGIES USED

### FRONTEND

1. React JS (useContext, useReducer)
2. React Router Hash Link
3. TailwindCSS
4. MUI Icons
